API:
The 'itemart' and 'gumpart' APIs return PNG images. Hue is an optional parameter. The 'cliloc' API returns a plaintext string. Input may be decimal (10) or hexadecimal (0xA).
http://uo.cx/api/v1/itemart/(itemid)[/hue]
http://uo.cx/api/v1/gumpart/(itemid)[/hue]
http://uo.cx/api/v1/cliloc/(number)

Examples:
http://uo.cx/api/v1/itemart/7939/0x489
http://uo.cx/api/v1/gumpart/60469/1152
http://uo.cx/api/v1/cliloc/500015
